## CouponApp

Käytetyt teknologiat:

-   JavaScript
-   React Native
-   Firebase realtime database
-   Prettier (koodin formatointiin)

Sovelluksen käynnistysohjeet

1. Kloonaa repository

```sh
git clone https://github.com/olgatyynela/couponapp.git
```

2. Aja komennot

```sh
npm install
```

```sh
npm start
```
